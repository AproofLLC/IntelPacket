# Core benchmarks

| Script | Purpose |
|--------|---------|
| `pnpm run bench` | Dataset compression / pipeline wall-clock (existing `compression-bench.ts`). |
| `pnpm run bench:50x` | Repeated engine timings. |
| `pnpm run bench:realistic` | **Synthetic realistic JSON** → full packet pipeline → determinism, verify, replay round-trip → writes `benchmark-results.json`, `BENCHMARK_REPORT.md`, and copies under `outputs/`. |
| `pnpm run bench:advanced` | Advanced CLI bench: entropy datasets, gzip/brotli baselines, p50/p95/p99, ops/sec, memory → `advanced/outputs/core-advanced-results.json`, `CORE_ADVANCED_REPORT.md`. Optional `--scale=10k\|100k\|1m`, `--iterations=N`, `--strict-scale`. |
| `pnpm run datasets:realistic` | Regenerate JSON fixtures in `datasets/realistic/` (deterministic generator; **synthetic data only**). |

## Reports

- [BENCHMARK_REPORT.md](./BENCHMARK_REPORT.md) — human-readable summary (regenerated by `bench:realistic`).
- [benchmark-results.json](./benchmark-results.json) — machine-readable metrics (same).
- [advanced/outputs/CORE_ADVANCED_REPORT.md](./advanced/outputs/CORE_ADVANCED_REPORT.md) — advanced bench (regenerated by `bench:advanced`).

## Determinism & replay

`bench:realistic` fails if:

- five consecutive `packet_hash` values differ for the same input;
- `verifyIntelPacket` is false;
- `createPacket(replay.normalized)` does not reproduce the original `packet_hash`.

## Non-goals

Benchmarks are not load tests, security proofs, or production SLO guarantees.
