# Advanced benchmark outputs

Generated by `pnpm run bench:advanced` (or `pnpm run bench:core:advanced`) in `@intelpacket/core`:

- `core-advanced-results.json`
- `CORE_ADVANCED_REPORT.md`

Files are overwritten each run. Do not treat as cryptographic proofs.

**CLI:** `--scale=10k|100k|1m`, `--iterations=N`, optional `--strict-scale` (use full requested row count; large scales can be slow or memory-heavy). Without `--strict-scale`, `100k` / `1m` modes may use fewer records — see JSON `cli.recordCount` vs `cli.scaleRequested`.
