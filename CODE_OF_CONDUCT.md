# Code of conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1. A full text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Project maintainers may remove, edit, or reject contributions or interactions that violate this code. Report concerns to the repository owners via GitHub issues or the contact route described in `SECURITY.md` when appropriate.
